# 🎯 Student Scoreboard App

A simple and interactive React application to manage and display student scores. Built using React (Vite) with a clean component-based architecture.

---

## 🚀 Features

* 📋 Display a list of students with their scores
* ✏️ Update student scores dynamically
* ➕ Add new students using a form
* ✅ Pass/Fail status based on marks

  * Pass: Score ≥ 40
  * Fail: Score < 40
* 🎨 Clean and minimal UI using pure CSS

---

## 🛠️ Tech Stack

* React (with Vite)
* JavaScript (JSX)
* CSS (Pure CSS, no frameworks)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/student-scoreboard.git
```

2. Navigate to the project folder:

```
cd student-scoreboard
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```

5. Open in browser:

```
http://localhost:5173/
```
